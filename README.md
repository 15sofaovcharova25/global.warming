# 🌍 Эко-бот Потепление

Telegram-бот для образования о глобальном потеплении, экологических привычках и викторин по защите планеты.

---

## 🤖 О боте

Эко-бот Потепление — это интерактивный помощник, который в игровой форме рассказывает о проблемах климата и дает практические советы, как каждый человек может помочь планете.

**Бот содержит:**
- 📊 **100 фактов** о глобальном потеплении по 10 темам
- 🌱 **60 советов** по экологии по 6 темам
- ❓ **150 вопросов** для викторины по 15 темам
- 🎮 **Удобное меню** с кнопками для навигации

## 🎮 Как пользоваться

1. **Начни диалог** — отправь команду `/start`
2. **Выбери раздел** — нажми на кнопку в главном меню:
   - `📊 Факт` — узнать интересные факты
   - `🌱 Совет` — получить полезные советы
   - `❓ Викторина` — проверить свои знания
   - `📋 Меню` — открыть навигацию
3. **Выбери тему** — кликни на интересующую тему из списка
4. **Получай контент** — читай факты, советы или отвечай на вопросы
5. **Изучай дальше** — используй кнопки под сообщениями:
   - `🔄 Ещё` — следующий факт/совет в той же теме
   - `📋 Другая тема` — вернуться к выбору темы
   - `🔙 Главное меню` — вернуться в начало

### В викторине:
- Выбери тему или "Все темы"
- Нажимай на кнопки с вариантами ответов
- Получай объяснение правильного ответа
- Используй кнопки `➡️ Следующий` для продолжения

## 🛠 Технологии

- **Python 3.10+** — основной язык программирования
- **pyTelegramBotAPI** — библиотека для работы с Telegram API
- **python-dotenv** — для загрузки токена из файла .env

## 🔧 Установка и запуск

### Пошаговая инструкция:

#### 1. Клонировать репозиторий
```bash
git clone https://github.com/твой-логин/eco-bot.git
cd eco-bot
2. Установить зависимости
bash
pip install -r requirements.txt
3. Получить токен бота
Открой Telegram

Найди @BotFather

Отправь команду /newbot

Следуй инструкциям (придумай имя боту, например Эко бот Потепление)

После создания BotFather пришлет токен — скопируй его

4. Создать файл .env
Создай файл .env в папке проекта и добавь туда:

text
BOT_TOKEN=твой_токен_сюда
(вместо твой_токен_сюда вставь токен, который дал BotFather)

5. Запустить бота
bash
python bot.py
Если всё сделано правильно, ты увидишь:

text
🚀 Эко-бот Потепление запущен!
✅ Имя: @твой_бот
📊 Фактов: 100 по 10 темам
🌱 Советов: 60 по 6 темам
❓ Вопросов: 150 по 15 темам
📋 Нажми Ctrl+C для остановки
6. Найти бота в Telegram
Открой Telegram

В поиске введи имя своего бота (которое дал BotFather)

Нажми Start или отправь команду /start
