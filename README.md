## 🤖 О боте

**Эко-бот Потепление** — это интерактивный помощник, который в увлекательной форме рассказывает о проблемах климата и даёт практические советы, как каждый человек может помочь планете.

```python
# Статистика бота
ФАКТЫ = 100     # интересных фактов по 10 темам
СОВЕТЫ = 60     # полезных советов по 6 темам
ВОПРОСЫ = 150   # вопросов для викторины по 15 темам
```

## ✨ Функционал

### 🎯 Главное меню

```
┌─────────────────────────────────────┐
│  🌍 ЭКО-БОТ ПОТЕПЛЕНИЕ              │
│  Выбери, что хочешь узнать:         │
├─────────────────────────────────────┤
│  📊 ФАКТ     │  🌱 СОВЕТ             │
├─────────────────────────────────────┤
│  ❓ ВИКТОРИНА │  📋 МЕНЮ              │
└─────────────────────────────────────┘
```

| Кнопка | Команда | Что делает |
|:------:|:-------:|------------|
| 📊 Факт | `/facts` | Показывает меню выбора темы для фактов |
| 🌱 Совет | `/tips` | Показывает меню выбора темы для советов |
| ❓ Викторина | `/quiz` | Показывает меню выбора темы для викторины |
| 📋 Меню | `/menu` | Открывает главное навигационное меню |

### 🔘 Кнопки управления

После получения контента доступны кнопки:

```
🔄 ЕЩЁ           → следующий в той же теме
📋 ДРУГАЯ ТЕМА   → вернуться к выбору темы
🔙 ГЛАВНОЕ МЕНЮ  → вернуться в начало
```

### 🎲 В викторине

```
✅ Правильно!    → получаешь объяснение
❌ Неправильно   → узнаёшь правильный ответ
➡️ СЛЕДУЮЩИЙ     → новый вопрос в той же теме
📋 СМЕНИТЬ ТЕМУ  → выбор другой темы
🔙 ГЛАВНОЕ МЕНЮ  → вернуться в начало
```

## 🔧 Установка

### 1️⃣ Клонировать репозиторий

```bash
git clone https://github.com/твой-логин/eco-bot.git
cd eco-bot
```

### 2️⃣ Создать виртуальное окружение (рекомендуется)

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### 3️⃣ Установить зависимости

```bash
pip install -r requirements.txt
```

### 4️⃣ Получить токен бота

1. Открой Telegram
2. Найди [@BotFather](https://t.me/botfather)
3. Отправь команду `/newbot`
4. Следуй инструкциям (придумай имя боту, например `Эко бот Потепление`)
5. После создания BotFather пришлёт токен — скопируй его

### 5️⃣ Создать файл `.env`

Создай файл `.env` в папке проекта и добавь туда:

```env
BOT_TOKEN=твой_токен_сюда
```

> ⚠️ **Важно:** Никогда не публикуй этот токен в открытом доступе! Файл `.env` уже добавлен в `.gitignore`
