# global.warming
Основная идея игры "Эко-квест: Спаси планету":

Игрок управляет уровнем CO2 в атмосфере через свои повседневные решения.

Механика:

Игра начинается с критического уровня CO2 (100%)

Каждый выбор игрока влияет на этот показатель

Экологичные решения снижают CO2

Вредные для природы решения повышают CO2

Цель:
Довести уровень CO2 до нуля (победа), не допустив его повышения до 200% (поражение)

Образовательный посыл:
Игра показывает, что даже маленькие бытовые решения (выключить свет, выбрать автобус вместо машины, сдать бумагу в переработку) в сумме реально влияют на климат. Чем больше экопривычек у человека, тем больше шансов спасти планету.

Аудитория:
Дети и подростки (простой интерфейс, понятные вопросы, мгновенная обратная связь)

Формат:
Текстовый квест с визуальным индикатором прогресса
